Tolkien Book Search

A small web application that lets users search and explore a list of books related to J.R.R. Tolkien.
Users can type book names (or parts of names) in a search box and instantly filter the displayed list to find specific books. This project demonstrates DOM manipulation, JavaScript logic, and dynamic search filtering.
